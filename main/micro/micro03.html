飛梭雷射治療的原理

飛梭雷射治療使用雷射在皮膚上創造許多細小的加熱區 域。雷射能量穿透到400到700微米 的皮膚深 度，啟動了膠原蛋白增生以及皮膚自我修復。每一個個小 的加熱區周邊都包圍著正常且結構完整的皮膚組織，治療 區域可在一天內快速癒合，因而沒有不良的副作用， 如開放性傷口或者嚴重紅腫脫屑等。這種分段式 ( fractional ) 的治療理念，大為降低以往傳統磨皮換膚引起 的手術後照顧不便與感染風險，使肌膚傷口可在 極短時間之內便可癒合，同時藉由身體自然的修復過程， 將缺陷受損肌膚加以排除，置換為更新、更健康、更緊實的組織。

Q1.你需要飛梭雷射治療嗎﹖

當你感受到皮膚出現疲憊、皺紋、色素不均、 毛孔粗大、青春痘疤凹凸不平等困擾時，飛梭 雷射必定是您最佳的選擇。不同於其他傳統磨 皮雷射傷害性高，飛梭雷射可以使用於全身任何 部位，甚包含眼睛周圍、頸部、胸部或手背 這類敏感細緻的肌膚。治療後的肌膚將看起來 更平滑、水亮，讓您將看起來更加年輕、健康 並充滿活力。

Q2.治療過程如何進行呢?

飛梭雷射治療前須先卸妝清潔，之後在以外用 麻醉藥膏塗敷約1小時，隨即於治療部位塗抹 飛梭雷射專用藍色導光劑與潤滑藥膏，提高治 療部位輪廓的對比性，以增加治療的準確度。 全臉的治療時間約20-30分鐘，頸部或手部等 處約10分鐘便可完成。

Q3.我需要做多少次治療呢?

基於飛梭雷射治療的分段式設計，每次治療僅 約20%的皮膚直接承受雷射能量，因此大多數 病患需要約5次的治療，間隔則為2周以上，確 切次數與間隔時間仍須經由醫師專業的諮詢診 斷。此外，飛梭雷射治療後的肌膚改善效果卻 是立即可見並且持續發生，一般最佳的改善結 果將於治療後2-3個月呈現。

Q4.治療前要注意什麼?

進行飛梭雷射治療之前，請您與醫師詳加討論 適合治療的肌膚部位、可預期的治療結果、接 續下來的轉變，以及療程的規劃。術前一個月 內避免過度的日曬或SPA，若有服用藥物，懷 孕，光敏感病史者，務必徵詢醫師意見。另治 療前2周內不得使用A酸 (如Retin-A )及A醇。

Q5.治療時有什麼感覺?

治療時會有能量進入皮膚的暫時性刺熱感，飛 梭雷射治療完畢後需冰敷15-30分鐘，接續的 1 小時內則如輕微日曬般的感受，幾乎無嚴重不適 。如果紅熱狀況持續，可以冰敷方式鎮定，並 安排回診檢視。

Q6.治療後要如何照護維持?

治療後5-7天內肌膚將呈現粉紅色調，此為皮 膚深部組織進行修復的正常現象。腫脹的情形 非常輕微，且於2-3內便消退。於治療完畢後 ，您即可立刻保養及上妝，大部分的病患於治 療後或治隔日便可正常上班。治療後的清潔 保養動作請輕柔，並著重保溼與防曬步驟，切 勿用力摩擦或進行去角質。

Q7.治療後的注意事項?

表皮的新生在治療後24小時內立即發生，在肌 膚修復的過程與注意事項︰


暫時性古銅色化 ( Bronzing )

依照治療過程的不同，您的皮膚將呈現3-14天 不等的暫時性膚色加深。


暫時性脫皮 ( Flaking )

您的好老化壞死組織將自然而然的脫落，取而代 之的是新生的健康肌膚。脫皮的情形如同日曬 後的反應，卻不會有痛感。加強保溼產的使 用可改善脫皮的現象。


暫時性脫皮 ( Sunscreen )

治療開始至之後6個月內，每日皆須使用防 曬品至少2次，以保護皮膚避免陽光曝曬。請 選用SPF至少30以上之滋潤型防曬乳，另頃戴 寬邊帽子以加強防護效果。


暫時性脫皮 ( Maintenance )

飛梭雷射治療雖然能幫助您撥回時鐘的指針， 但是它無法完全停止老化的過程。若要維持治 療結果，病患必須維持良好生活習慣，減少壓 力並且避免太陽曝曬。這些步驟都是幫助您保 持年輕外表的不二法門。
